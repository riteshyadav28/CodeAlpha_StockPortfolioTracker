# Stock Portfolio Tracker

## Description

This project is a simple Stock Portfolio Tracker developed using Python as part of the CodeAlpha Python Programming Internship.

The user enters stock names and quantities. The program calculates the total investment using predefined stock prices and saves the results in a CSV file.

## Features

- Predefined stock prices
- User input for stock names and quantities
- Investment calculation
- CSV file generation
- Simple console interface

## Technologies Used

- Python 3

## Concepts Used

- Dictionary
- Loops
- File Handling
- CSV Module
- Conditional Statements

## How to Run

```bash
python portfolio.py
```
